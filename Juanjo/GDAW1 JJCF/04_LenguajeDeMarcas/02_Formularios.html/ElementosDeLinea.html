Atributos de etiqueta a

href
URL de destino
Modificadores:
  • mailto: lo usamos para indicar que el enlace se refiere a una direccion de correo electrónico 
    <a href="mailto.contacto@miempresa.com">mandame un email</a>

hrefLang
Idioma del documento de destimno
ping
Permite indicar una o más URL´s sobre las que se realizaran peticiones
rel
Determina la relacion entre el enlace y el recurso al que hace referencia
target
Permite indicar donde abrir la URL del enlace
Tipos de target:
  • _self: Abre en mi misma ventana/pestaña
  • _blank: Abre en una ventana/pestaña nueva
  • _parent: Abre en el contexto de navegación padre del actual
  • _top: Abre en el contexto de navegación más alto
