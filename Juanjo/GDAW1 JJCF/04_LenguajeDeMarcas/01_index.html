Glosario

Tema 1

  1. DOCTYPE html: indicamos que el tipo de documento es html
  2. html lang="es": etiqueta de apertura del documento
  3. head: abrimos la cabecera del documento
  4. meta charset="utf-8": metadato (información añadida, se puede utilizar para hacer cracking). charset es el conjunto de caracteres con el que voy a trabajar en ese documento
  5. title: título del documento. Aparece en la barra del título de la ventana o en la pestaña del navegador
  6. div: se emplea para definir un bloque de contenido o sección de la página, para poder aplicarle diferentes estilos o incluso realizar operaciones sobre ese bloque
  7. body: cuerpo del documento. la parte visible
  8. header: encabezado
  9. h1: encabezado de nivel 1 o de mayor relevancia, solo puede haber uno salvo una excepción
  10. p: párrafo
  11. br: quiebre de línea, causa que el texto continúe en una línea nueva
  12. nav:Texto, contenido real, representa una sección de una página cuyo propósito es proporcionar enlaces de navegación
  13. ul: Lista sin orden
  14. li: List item. Elemento de la lista
  15. ol: Lista ordenada
  16. article: Especifica contenido independiente y autónomo y debe tener sentido por sí mismo
  17. dl: Lista de definiciónes o descripciones. Agrupa (dt) y (dd)
  18. dt: Especifica un término en una descripción o lista de definciones, debe utilizarse dentro de una lista de definiciones(dl)
  19. dd: Aporta detalles acerca de la definición de un término precedente (dt) dentro de una lista de definciones (dl)
  20. footer: Pie de pagina
